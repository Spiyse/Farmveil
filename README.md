A cozy, minimal top-down farming game made with Love2D.

## How to Play

### Controls
- **1** - Select Hoe
- **2** - Select Seeds
- **3** - Select Watering Can
- **4** - Select Hand
- **Left click** - Use current tool
- **Left click** - to move around the camera
- **scroll wheel** - to zomm in and out

## Running the Game

Make sure you have [Love2D](https://love2d.org/) installed, then:

go into the project through *vscode* or *cd* into it.

```bash
love .
```
